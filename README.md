## 安装
### 压缩包上传方式（推荐）
下载最新发行版zip压缩包到本地

进入博客后台-外观-主题-安装

选择本地上传，将压缩包上传上去

等待提示安装完成

### 在线拉取方式
进入博客后台-外观-主题-安装

选择远程下载

远程地址填写 https://github.com/cross266/halo-theme-iwtwin.git

点击下载，等待下载完成

**主题个性化的配置请在 Iwtwin 的主题设置中设置！**
